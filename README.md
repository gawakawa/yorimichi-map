# 寄り道マップ

**URL**: <https://yorimichi.i0ta.dev>

- 寄り道スポットを含めた経路探索アプリ

## 特徴

- **AI による寄り道スポット提案** - Gemini がユーザーの要望を理解し、最適な寄り道スポットを提案
- **チャット形式の操作** - 自然な対話で経路探索と寄り道スポットを検索
- **リアルタイム交通情報対応** - 現在の交通状況を考慮した最適ルートを計算
- **Google Maps 連携** - ワンタップでナビアプリを起動
- **インタラクティブな地図表示** - Leaflet + OpenStreetMap によるルート可視化
